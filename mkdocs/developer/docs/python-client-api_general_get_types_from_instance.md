# get\_types\_from\_instance

**get\_types\_from\_instance(instance\_type)**

Get the connector types from an instance type

**param:**

**instance\_type** - the search type of the instance

**return:**

**tuple** - (from\_type, parent\_type)

a tuple with the from\_type and the parent\_type. The from\_type is

the connector type and the parent type is the search type of the

parent of the instance
